# 宁夏理工学院NIT-LUG

## 关于我们

宁夏理工学院 Linux 用户组（NIT Linux User Group, NIT LUG）是由一群对 GNU/Linux 及开源技术充满热情的学生组成的开放性组织。旨在通过学习和分享，宣传自由软件精神，推动 Linux 操作系统以及自由软件生态的应用和普及，帮助成员掌握 Linux 技能，搭建信息交流共享的平台。
